Installation Guide

Follow these steps to set up and launch your Mobile Boat Detailing website.

1. Extract & Setup

Download the project ZIP file and extract it to your desired local directory. Recommended folder structure:

/boat-detailing/
├── /css/ (Styling)
├── /js/ (Interactivity)
├── /images/ (Optimized Assets)
├── index.html (Home)
└── /documentation/

2. Code Editor

Open the folder in a high-quality editor like VS Code. This allows you to easily edit HTML, CSS, and JS files with syntax highlighting.

3. Local Run

Use the Live Server extension in VS Code or simply double-click index.html to view the project in your browser.

Note: Using a local server is recommended for better handling of assets and routing.

4. Deployment

Upload the extracted files via FTP/SFTP to your server's public_html folder or deploy using GitHub Pages, Netlify, or Vercel.

Common Issues & Fixes

  • 404 Images: Ensure image paths are relative and the /images folder exists.
  • Layout Breaking: Check if you have modified the CSS variables incorrectly.
  • Scripts not loading: Ensure scripts are linked before the closing </body> tag.

Customization Guide

Tailor the branding and appearance of your website to match your boat detailing business.

Branding & Colors

Modify the primary colors in index.css or the <style> block using CSS variables:

:root {
  --primary: #00d2ff; /* Ocean Blue */
  --primary-dark: #00a8cc;
  --bg: #0a0c10; /* Premium Dark */
}

💡 Tip: Use a "Before / After" approach. Save your file, refresh the browser, and check the changes immediately.

Typography

We use Outfit from Google Fonts. To change it, update the link in the <head> and the font-family in CSS.

Images

Replace assets in the /images folder.
Recommended Sizes:
Hero: 1920x1080px (WebP)
Gallery: 800x600px

Text Content

Edit headings, paragraphs, and buttons directly in the HTML. Look for <!-- TODO: EDIT CONTENT --> comments in the source code to find customizable sections quickly.
